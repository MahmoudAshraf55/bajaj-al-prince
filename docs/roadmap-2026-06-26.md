# BAJAJ AL PRINCE — Roadmap & Ideas (26 June 2026)

## ✅ Completed
- Public website with 3D Hero, bilingual EN/AR, booking, marketplace
- Admin portal: Dashboard, CRM, Vehicles, Products, Cashier, Work Orders
- WhatsApp integration: QR connect, auto-reminders, anti-ban, templates, schedules
- Foundation: Soft delete, Decimal precision, tenantId, API versioning, XSS sanitization
- Email service (nodemailer) integrated with booking flow
- CI/CD pipeline (GitHub Actions)
- API unit tests (Playwright)
- Full documentation: SYSTEM_RULES, PROJECT_RULES, GIT_GOVERNANCE, VERSIONING_STRATEGY, RELEASE_MANAGEMENT, BACKUP_AND_RECOVERY

## 🚀 Next Features (Priority Order)

### Phase 1: Security & Monitoring
- [ ] **2FA (OTP)** — Send OTP via WhatsApp/Email for admin login
- [ ] **Sentry activation** — `npm install @sentry/nextjs`, init in layout
- [ ] **Rate limit tuning** — Per-endpoint limits based on usage patterns
- [ ] **Admin session management** — View active sessions, force logout

### Phase 2: Inventory & Stock Management
- [ ] **Barcode scanning** — Scan barcodes for product entry/exit
- [ ] **Stock alerts** — Low stock notifications (WhatsApp/email)
- [ ] **Supplier management** — Supplier profiles, purchase orders
- [ ] **Warehouse zones** — Organized storage locations
- [ ] **Inventory audit** — Physical vs system stock reconciliation

### Phase 3: Cashier & POS
- [ ] **Quick sale POS** — One-click sale with receipt printing
- [ ] **Receipt printer** — Thermal printer integration (ESC/POS)
- [ ] **Invoice generation** — PDF invoices with logo
- [ ] **Daily/Monthly reports** — Sales summaries, profit/loss
- [ ] **Expense categories** — Categorized expense tracking

### Phase 4: Customer Portal
- [ ] **Customer login** — Simple login via phone + OTP
- [ ] **Service history** — View past bookings and work orders
- [ ] **Invoice download** — PDF invoices for completed services
- [ ] **Feedback system** — Rate service after completion
- [ ] **Loyalty points** — Points per service visit

### Phase 5: Advanced Features
- [ ] **Push notifications** — Web push for admin on new bookings
- [ ] **Export to Excel/CSV** — For customers, bookings, transactions
- [ ] **Admin dark/light toggle** — Theme switching
- [ ] **Docker Compose** — Containerize Next.js + WhatsApp service + PostgreSQL
- [ ] **Multi-branch management** — Branch switching in admin
- [ ] **WhatsApp session backup** — Auto-backup `.baileys_auth/`
