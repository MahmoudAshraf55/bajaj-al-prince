# دليل استخدام ChatGPT — ChatGPT User Guide

## ما هو ChatGPT؟
ChatGPT هو مساعد ذكاء اصطناعي يفهم الأسئلة والطلبات باللغة العربية والإنجليزية. يمكنك استخدامه لكتابة الكود، تصحيح الأخطاء، التعلم، أو حتى التخطيط لمشاريع مثل هذا الموقع.

---

## 1. كيف تكتب طلب (Prompt) صحيح

### الطلب الجيد = النتيجة الجيدة

| طلب ضعيف (سيء) | طلب قوي (جيد) |
|---|---|
| "اعمل لي موقع" | "أريد موقع Next.js يعرض قطع غيار دراجات نارية مع صفحة للحجز" |
| "الكود مش شغال" | "يظهر خطأ `Property 'x' does not exist` في ملف `Hero.tsx` السطر 45" |
| "حسن لي التصميم" | "أريد زيادة التباين بين النص والخلفية في قسم Contact، استخدم Tailwind" |

### عناصر الطلب الناجح
```
[السياق] + [المشكلة/الهدف] + [التفاصيل التقنية] + [التنسيق المطلوب]
```

**مثال:**
> "أعمل على مشروع Next.js 16 + TypeScript + Tailwind CSS. المشكلة: الـ 3D model يظهر في منتصف الشاشة وأريده على اليمين. الملف: `MotorcycleScene.tsx`. الكاميرا حالياً عند `position={[3, 1.5, 4]}` والـ model group عند `position={[0, 0.2, 0]}`. أريد الموديل يظهر على يمين الشاشة (أمام نص العنوان)."

## 2. أوامر مفيدة أثناء البرمجة

### أ. شرح الكود
```
"اشرح لي هذا الكود سطر بسطر: [تلصق الكود هنا]"
```

### ب. تصحيح الأخطاء
```
"يظهر لي هذا الخطأ: [error message]
في هذا الملف: [path]
محتوى الملف: [paste code]"
```

### ج. إعادة كتابة الكود بطريقة أخرى
```
"أعد كتابة هذا الكود باستخدام [React hooks / GSAP / Framer Motion] بدلاً من [الطريقة الحالية]"
```

### د. إنشاء خطة تنفيذ
```
"أريد إضافة [feature] للمشروع. اكتب لي خطة تنفيذ خطوة بخطوة."
```

## 3. التواصل مع Cascade (IDE AI)

Cascade هو AI مدمج في IDE (مثل Cursor/Windsurf). يمكنه قراءة وتحرير ملفاتك مباشرة.

### أوامر Cascade المفيدة
| الأمر | الغرض |
|---|---|
| `"اقرأ لي الملف [path]`" | يقرأ محتوى ملف معين |
| `"عدل السطر [X] في [file] ليصبح [code]`" | تعديل دقيق |
| `"ابحث لي عن [term] في المشروع`" | يبحث في كل الملفات |
| `"شغل لي السيرفر`" | يشغل `npm run dev` |
| `"اعمل build`" | يشغل `npm run build` ويتحقق من الأخطاء |

### أهم نصيحة
> **أخبر Cascade دائماً بالسياق الكامل.**
>
> بدلاً من: "غير اللون"
>
> قل: "في ملف `Services.tsx`، غير خلفية الـ section من `bg-secondary/30` إلى `bg-background` عشان الـ 3D model ما يبينش من تحت"

## 4. أسئلة شائعة

### هل يمكنني تعلم البرمجة من ChatGPT؟
نعم، لكن استخدمه كـ "معلم" لا كـ "بديل". اطلب الشرح، جرب الكود بنفسك، عدل فيه.

### لماذا يعطيني ChatGPT كود قديم؟
بياناته تتوقف عن تاريخ معين. للحصول على أحدث API، اطلب منه قراءة التوثيق الرسمي:
> "اقرأ لي هذا الرابط: [docs link] ثم اطبقه"

### كيف أجعله يفهم مشروعي بالكامل؟
1. استخدم `attach` أو `reference` لملفات المشروع
2. ابدأ بالسياق: "أعمل على مشروع Next.js 16 App Router..."
3. اذكر المكتبات المستخدمة: "نستخدم GSAP + Three.js + Tailwind 4"

## 5. قالب سريع لطلبات البرمجة

انسخ والصق هذا القالب واملأ الفراغات:

```
المشروع: [Next.js / React / Vue / إلخ]
المكتبات: [اسم المكتبات]
الملف: [المسار الكامل للملف]
المشكلة/الهدف:
[اشرح ما تريده بالتفصيل]

الكود الحالي:
```
[الصق الكود هنا]
```

النتيجة المطلوبة:
[اشرح ما يجب أن يحدث]
```

## 6. مصطلحات أساسية

| المصطلح | المعنى |
|---|---|
| **Prompt** | الطلب الذي تكتبه للـ AI |
| **Context** | السياق — المعلومات التي يحتاجها AI لفهم وضعك |
| **Token** | وحدة نصية يقرأها AI (كلمة أو جزء من كلمة) |
| **Hallucination** | "هلوسة" — عندما يخترع AI معلومات خاطئة |
| **Debug** | البحث عن وإصلاح الأخطاء |
| **Refactor** | إعادة تنظيم الكود دون تغيير وظيفته |

## ملخص سريع

| نصيحة | الخلاصة |
|---|---|
| كن محدداً | كلما زادت التفاصيل، زادت دقة النتيجة |
| أرفق الأخطاء | الصق رسالة الخطأ كاملة مع اسم الملف ورقم السطر |
| أرفق الكود | الصق الكود المرتبط بالمشكلة |
| اختبر بنفسك | لا تعتمد على AI 100% — راجع وفهم |
| استمر في المحاولة | إذا لم تعجبك الإجابة، اشرح أكثر أو اطلب تعديلاً |
