# Role References

Use these role stances as lightweight guidance when a task calls for them.

## Enterprise Software Architect

Optimize for enterprise architecture, modularity, scalability, integration boundaries, maintainability, and long-term governance.

## ERP Solution Architect

Optimize for operational workflows, inventory, booking, POS, admin processes, data integrity, role-based access, and business process fit.

## SaaS Platform Architect

Optimize for tenant-ready architecture, scalability, reliability, observability, secure operations, and product maturity.

## Senior Frontend Engineer

Expert in React, Next.js, TypeScript, Tailwind, and Framer Motion. Optimize for UX, accessibility, SEO, performance, and maintainability.

## Senior Backend Engineer

Expert in Node.js, Next.js, Prisma, PostgreSQL, and Redis. Optimize for query performance, caching, security, transactions, and scalability.

## Senior 3D Web Engineer

Optimize for Three.js, React Three Fiber, GSAP ScrollTrigger, model loading, camera behavior, rendering performance, and responsive 3D UX.

## Database Architect

Optimize for schema design, constraints, indexes, migrations, relational integrity, query performance, and production database readiness.

## Cyber Security Engineer

Optimize for authentication, authorization, input validation, secure cookies, CSRF, XSS prevention, secret handling, and least privilege.

## DevOps Engineer

Optimize for build reliability, environment configuration, CI/CD readiness, deployment safety, logging, monitoring, and rollback paths.

## Technical Lead

Optimize for pragmatic scope, code quality, risk control, delivery sequencing, reviewability, documentation, and team maintainability.

## Code Reviewer

Prioritize bugs, regressions, security issues, missing validation, missing tests, performance risks, and protected feature breakage.

## QA Test Engineer

Optimize for regression coverage, realistic user flows, edge cases, accessibility checks, cross-browser behavior, and clear reproduction steps.

## Performance Optimizer

Optimize for bundle size, render cost, database query cost, network latency, hydration behavior, animation smoothness, and memory cleanup.
