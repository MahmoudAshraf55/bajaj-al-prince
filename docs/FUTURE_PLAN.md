# خطة المميزات المستقبلية — Bajaj El Prince

> المميزات دي مش هنشتغل عليها دلوقتي. لما تقول "ابدأ في [اسم الميزة]"، هنبدأ.

---

## 1. تصدير Excel/PDF للتقارير
- زر "تصدير Excel" في صفحة المحاسبة (P&L)
- زر "تصدير PDF" لكشف الدخل
- تصدير قائمة الجرد
- تصدير حركات المخزون
- المكتبات: `xlsx` موجودة بالفعل، محتاجة `jspdf` + `jspdf-autotable`

## 2. مركز الرسائل الموحد (Inbox)
- توحيد رسائل الحجوزات + واتساب + استفسارات الموقع في صندوق واحد
- تبويبات: الكل / الحجوزات / واتساب / الموقع
- إمكانية الرد من نفس الشاشة
- فلترة ومتابعة

## 3. التقارير المتقدمة
- PDF للتقارير المحاسبية (P&L, إيرادات, مصروفات)
- تقارير الفروق (Variance reports) للجرد
- رسم بياني للإيرادات والمصروفات
- فلترة متقدمة (بالمستخدم، طريقة الدفع، الفئة)

## 4. إشعارات الموبايل (Push Notifications)
- إشعارات للموظفين عند حجز جديد
- إشعارات للعملاء عند تغيير حالة الحجز
- مكتبة: Firebase Cloud Messaging أو OneSignal

## 5. الفواتير الإلكترونية (حكومة)
- التكامل مع منظومة الفواتير الإلكترونية المصرية
- توقيع إلكتروني للفواتير
- كود الاستيفاء للمنتجات

---

> **ملحوظة:** الخطة قابلة للتعديل — تقدر تضيف أو تشيل أي ميزة في أي وقت.
